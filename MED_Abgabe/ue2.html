Gelesene Header-Informationen

Erklärung und Berechnung

  • Dateigröße Gibt an, wie groß die gesamte BMP-Datei in Bytes ist (Header + Pixeldaten)
  • Datenoffset Legt fest, ab welcher Byte-Position im File die eigentlichen Bildpixel beginnen (meist 54)
  • Breite Die Anzahl der Pixel pro Zeile (horizontale Auflösung)
  • Höhe Die Anzahl der Pixel pro Spalte (vertikale Auflösung)
  • Bits pro Pixel Gibt an, wie viele Bits ein Pixel besitzt (Farbtiefe), z.B.
    • 24 Bit = 3 Byte (RGB)
    • 32 Bit = 4 Byte (RGBA)

  • Bildgröße Die Größe der reinen Pixeldaten in Bytes (Breite × Höhe × Bytes pro Pixel)=1555254
